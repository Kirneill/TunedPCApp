## v1.0.16 - UI/UX redesign integration

This release ships the new scalable UI/UX across the main app while keeping existing optimization behavior unchanged.

### What's new
- Added a persistent left sidebar for fast section navigation.
- Upgraded dashboard visuals with a cleaner red/black control-center style.
- Improved hierarchy and readability in the title bar and primary panels.
- Reworked game selection UX for growth:
  - search and install-status filters
  - quick picks section
  - scalable all-games list with batch select/clear
- Added clearer install-status indicators in game cards and lists.
- Fixed account dropdown layering so the email menu renders correctly above content.

### Behavior impact
- No optimization logic changes in this release.
- All changes are UI/UX only.
