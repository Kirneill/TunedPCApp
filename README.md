# TunedPC - PC Gaming Optimizer

One-click optimization for competitive FPS gaming. Tunes your Windows settings, game configs, GPU, and BIOS for maximum FPS and minimum input lag.

## What It Does

- **Game Settings** — Competitive configs for Fortnite, Valorant, CS2, COD, and more
- **Windows Optimization** — Disables background bloat, optimizes power plans, GPU scheduling
- **GPU Driver Profile** — One-click NVIDIA/AMD driver tuning
- **GPU Overclocker** — Safe GPU overclocking with Auto-Tune (Pro)
- **BIOS Optimizer** — Hardware-level CPU/RAM tuning matched to your system (Pro)

All changes are safe and reversible. Game settings back up before modifying. GPU overclocks reset on reboot.
