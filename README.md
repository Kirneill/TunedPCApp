# TunedPCApp (Public Releases)

TunedPCApp is a Windows optimization utility focused on reducing background overhead and improving gameplay consistency.

This repository is for **installer and update binaries only**.
The source code is private.

## Download

1. Open the [Releases](https://github.com/Kirneill/TunedPCApp/releases) page.
2. Download the latest `SENSEQUALITY-Optimizer-Setup-<version>.exe`.
3. Run the installer as Administrator.

## Quick Start for PC Gamers

1. Close your game, launcher, overlays, and browser tabs.
2. Launch TunedPCApp as Administrator.
3. Create a system backup in the app before applying any changes.
4. Apply the recommended optimization profile first.
5. Reboot your PC after applying major changes.
6. Test with your usual game and benchmark scene.
7. Keep settings that improve frametime stability; revert anything that hurts your setup.

## Auto Updates

- You only manually download the installer once.
- The app checks this repo's release feed for updates.
- Future updates download in-app when a new version is published.

## Safety and Rollback

- Different hardware, BIOS, drivers, and Windows builds behave differently.
- Always test changes on your own system.
- Use the app's backup/restore flow if you want to roll back.

## Troubleshooting

- If Windows SmartScreen appears, verify publisher/release details before continuing.
- If an update fails, install the latest `.exe` manually from Releases.
- If a tweak causes issues, restore from backup and reboot.
